## Cosa Sono gli AI Agents?

Gli **AI Agents** sono AI "potenziati" che possono:

‚úÖ **Agire autonomamente** per completare obiettivi
‚úÖ **Usare strumenti** (browser, terminale, API, file system)
‚úÖ **Pianificare** una sequenza di azioni
‚úÖ **Verificare** i risultati e correggere errori

### Come Funzionano

Un AI Agent:
1. **Riceve un obiettivo** (non solo un prompt)
2. **Pianifica** i passi necessari
3. **Esegue** le azioni usando tool esterni
4. **Verifica** i risultati
5. **Corregge** se necessario
6. **Completa** il task autonomamente

**Esempio pratico:**

Tu: "Crea un sito web portfolio con 3 pagine e deployalo"

  1. Crea struttura HTML/CSS

## Tutorial: Crea Il Tuo Primo AI Agent

Vediamo come creare un semplice AI Agent con **AutoGPT**.

### Requisiti

- Python 3.10+
- Account OpenAI (API key)
- Git

### Step 1: Installazione

```bash

cd AutoGPT

# Installazione dipendenze
pip install -r requirements.txt
```

### Step 2: Configurazione

Crea il file `.env`:

```bash
OPENAI_API_KEY=sk-your-api-key-here
```

### Step 3: Primo Agent

Crea `my_agent.py`:

```python
from autogpt.agent import Agent
from autogpt.config import Config

# Configurazione
config = Config()

# Crea agent
agent = Agent(
    ai_name="WebResearcher",
    ai_role="Research assistant specializing in tech topics",
    ai_goals=[
        "Search for latest 3D printing news",
        "Summarize top 5 findings",
        "Save results to markdown file"
    ]
)

# Esegui
agent.run()
```

### Step 4: Esecuzione

```bash
python my_agent.py
```
