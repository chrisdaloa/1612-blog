# Claude Editorial Instructions

Quando scrivi un articolo:

1. Leggi SEMPRE:
   - editorial-profile.md
   - christian-opinions.md

2. Recupera almeno 3 articoli simili da:
   /ai/style-examples/

3. Mantieni:
   - tono naturale
   - imperfezioni realistiche
   - opinioni pragmatiche

4. Evita:
   - tono AI
   - struttura troppo perfetta
   - liste inutili
   - entusiasmo artificiale

5. Gli articoli devono sembrare:
   scritti da un appassionato reale.

6. Se appropriato:
   inserisci dubbi realistici o considerazioni personali.

7. Non inventare esperienze dirette non confermate.

8. Usa markdown Hugo compatibile.

9. Genera sempre:
   - titolo
   - description
   - tags
   - slug

10. Gli articoli devono essere pronti per:
    /content/posts/

11. A fine elaborazione forniscimi un prompt da passare a gemini per la creazione della copertina dell'articolo

## Frontmatter

Seguire SEMPRE:
- /ai/frontmatter-rules.md
- /ai/templates/post-template.md

Il formato deve essere compatibile con Hugo e coerente con il repository esistente.

## Affiliate system

Quando scrivi articoli, devi seguire il sistema affiliati definito in:

- /ai/affiliate-policy.md
- /ai/affiliate-triggers.md

Questi file definiscono:
- quando inserire link affiliati
- come devono essere scritti
- limiti e comportamento

La logica dettagliata NON è qui, ma nei file dedicati.
