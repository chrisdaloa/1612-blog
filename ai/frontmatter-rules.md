# Hugo Frontmatter Rules

## Regole generali

- usare sempre YAML frontmatter
- mantenere coerenza con gli articoli esistenti
- usare date formato YYYY-MM-DD

---

## Draft

Gli articoli devono essere creati inizialmente con:

draft: true

---

## Tags

- usare tag realistici
- evitare troppi tag
- preferire 3-6 tag

---

## Categories

Usare categorie già esistenti nel sito.

---

## Cover

Quando possibile generare:

cover:
    image:
    alt:
    caption:
    relative: false

---

## Description

La description:
- deve sembrare naturale
- non SEO spam
- massimo 150-160 caratteri
