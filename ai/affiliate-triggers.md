# Affiliate Triggers — 1612.it

Questo file definisce quando inserire link di affiliazione Amazon negli articoli.

---

# REGOLA BASE

Usare link Amazon quando il contenuto riguarda prodotti reali che un lettore potrebbe acquistare o replicare.

I link devono essere sempre contestuali e mai forzati.

---

# HARDWARE

Usare affiliate link quando si parla di:

- mini PC
- NAS
- router
- switch di rete
- componenti hardware in generale

---

# MAKER / DIY

Usare affiliate link quando si parla di:

- ESP32
- Arduino
- alimentatori
- strumenti da banco
- elettronica DIY
- componentistica

---

# STAMPA 3D

Usare affiliate link quando si parla di:

- stampanti 3D
- resine
- accessori per stampa 3D
- upgrade hardware
- materiali e strumenti correlati

---

# SMART HOME

Usare affiliate link quando si parla di:

- sensori
- hub domotici
- dispositivi IoT
- automazione domestica

---

# REGOLE DECISIONALI

Prima di inserire un link, verificare:

1. Esiste un prodotto reale acquistabile collegato al tema?
2. Il lettore potrebbe voler replicare l’uso o l’acquisto?
3. Il prodotto migliora realmente la comprensione dell’articolo?

Se tutte le risposte sono sì:
→ inserire link Amazon

Se anche una risposta è no:
→ NON inserire link

---

# QUANDO NON USARE AFFILIATE

NON inserire link quando:

- l’articolo è teorico o concettuale
- non esiste un prodotto specifico rilevante
- l’inserimento sarebbe artificiale o forzato
- il contenuto è generale o informativo senza oggetti concreti

---

# OBIETTIVO

Le affiliazioni devono emergere naturalmente dal contesto tecnico,
come suggerimenti utili per il lettore e non come contenuto promozionale.
